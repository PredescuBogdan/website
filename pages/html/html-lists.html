Definition

There are 3 types of lists in HTML: unordered lists, ordered lists and description lists. We use the <ul> tags to define an unordered list and the <li> tags to define the list terms. We might also use type attributes (disk-default, circle, square) inside the <ul> and <li> tags. Ordered lists are defined with the <ol> tags and their terms with the <li> tags. We might also use types attributes (1-default, A, a, I, i) for ordered lists to define the type of the marker. A Description list is a list where each term have a description inside the list of terms. We use <dl> tags for the description list, <dt> tags for the term (name) and <dd> tags to define the data (description).

Syntax

Unordered Lists:

<ul></ul> - defines an unordered list

<li></li> - defines a list term

Ordered Lists:

<ol></ol> - defines an ordered list

<li></li> - defines a list term

Description Lists:

<dl></dl> - defines a description list

<dt></dt> - defines the term (name)

<ul></ul> - defines the data (description)

Example

Unordered List With Default Bullets

Code:

<ul>
   <li>1 A.M</li>
   <li>2 A.M</li>
   <li>3 A.M</li>
</ul>

Unordered List With Default Bullets

  • 1 A.M
  • 2 A.M
  • 3 A.M

Ordered List With Uppercase Letters

Code:

<ol type="A">
   <li>Morning</li>
   <li>Afternoon</li>
   <li>Evening</li>
</ol>

Ordered List With Uppercase Letters

  1. Morning
  2. Afternoon
  3. Evening

Description List

Code:

<dl>
   <dt>Smoking</dt>
   <dd>- bad habbit</dd>
   <dt>Jogging</dt>
   <dd>- good habbit</dd>
</dl>

Description List

Smoking
- bad habbit
Jogging
- good habbit

List Group With Contextual Classes

Code:

<ul class="list-group">
   <li class="list-group-item list-group-item-success">First book</li>
   <li class="list-group-item list-group-item-info">Second book</li>
   <li class="list-group-item list-group-item-warning">Third book</li>
   <li class="list-group-item list-group-item-danger">Fourth book</li>
</ul>

List Group With Contextual Classes

  • First book
  • Second book
  • Third book
  • Fourth book
