Definition

Images are defined with the <img> tag and does not have a closing tag. We can add a lot of images formats such as .gif, .png, .jpeg, .jpg, .bmp. To link an image to a document we have to inset the <img> tag inside <a> tags. The <img> tag requires two atributes (src and alt). Src defines defines the path to the file that contains the image(image URL). Alt defines the text that will be showed by the navigators that are not working in the graphical mode. We can also use more attributes like low src, width, height, border.

Syntax

<img src="url" alt="some_text"> - Insert image

Example

Images in another folder

It is very common on the web to store images in a folder:

Code:

<img class="img-responsive img-thumbnail" src="/images/pig.jpg" alt="pig">
<img class="responsive img-thumbnail" src="/images/linux.png" alt="linux">
