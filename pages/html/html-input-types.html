Definition

The HTML <input> tags are variables (areas) who can receive inserted text informations from the user of the form. We can use a lot of attributes like text, password, submit, radio, checkbox, date, button, reset. <input> elemnts are used inside <form> tags in order to declare input controls. All of these are used for different input types. Password inputs are used for password fields, checkbox inputs are used for multiple choices, radio inputs are used for single answer choices and so on.

Syntax

<input type="text"> - text input

<input type="password"> - password input

<input type="radio"> - radio input

<input type="checkbox"> - checkbox input

<input type="submit"> - submit input

<input type="button"> - button input

<input type="number"> - number input

In HTML 5 we have many more input types.

Example

Input types: text and password

Code:

<form role="form">
   <div class="form-group">
      <label for="usr">Full Name:</label>
      <input type="text" class="form-control" id="usr">
   </div>
   <div class="form-group">
      <label for="pwd">Password:</label>
      <input type="password" class="form-control" id="pwd">
   </div>
</form>
