Definition

HTML paragraphs are defined with the <p> tags. When browsers meet a <p> tag, they start a new line. For paragraphs we can use some align attributes (center, left, right). Line breaks are defined with the <br> tag. Line breaks are useful when we want a new line without starting a new paragraph. The preformatted text is defined with the <pre> tags.

Syntax

<p> - defines a paragraph

<br> - insert a single line break

<pre> - defines pre-formatted text

Example

Paragraphs

Code:

<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>
<p>This is the third paragraph.</p>
