Definition

An HTML structure for creating rows and columns on a Web page. You can think of it as a way to describe and display data that would make sense in spreadsheet software. Every table it is composed from the container table defined with the <table> tags, divided into table rows, table data and table headings. Table rows are defined with the <tr> tags, table data with the <td> tags and table headings with the <th> tags. The whole text is regular and left-aligned by default in the <td> elements and bold and centered by default in the <th> elements. Inside the <td> tag, we can define how many columns cell will span using colspan="no of columns" and how many rows cell will span using rowspan="no of rows".

Syntax

<table></table> - table

<tr></tr> - table row

<th></th> - table heading

<td></td> - table data

Example

Table (bordered table with contextual classes) :

Code:

<table class="table table-bordered">
    <thead>
      <tr>
        <th>Firstname</th>
        <th>Lastname</th>
        <th>Email</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Jill</td>
        <td>Smith</td>
        <td>jill.smith@gmail.com</td>
      </tr>
      <tr class="info">
        <td>Eve</td>
        <td>Jackson</td>
        <td>eve.jackson@gmail.com</td>
      </tr>
      <tr>
        <td>Donald</td>
        <td>Garcia</td>
        <td>donald.garcia@gmail.com</td>
      </tr>
    </tbody>
  </table>

Bordered table with contextual classes
