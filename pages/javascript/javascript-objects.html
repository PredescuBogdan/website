Definition

Any function in JavaScript can be used to create custom object classes. Objects are variables which can contain many values. For example we can assign to the glass, which is an object 3 values like: size, weight and color. In conclusion glass will be a variable with 3 values (var glass = {size:"small", weight:200, color:"white"};). Objects properties and methods can be accessed.

Syntax

var object = {}; - creating an object

objectName.propertyName - accessing object properties

objectName.methodName() - accessing object methods

Example

Object Definition

Creating a JavaScript Object

Code:

<p id="demo"></p>

<script>
var person = {name:"Bogdan", age:22, eyeColor:"blue"};

document.getElementById("demo").innerHTML =
person.name + " is " + person.age + " years old.";
</script>

Accessing Object Properties

Code:

<p id="demo2"></p>

<script>
var site = {
    txt1: "My website",
    txt2 : "cool",
};
document.getElementById("demo2").innerHTML =
site.txt1 + " " + "is" + " "+ site.txt2;
</script>

Accessing Object Properties

Accessing Object Methods

Code:

<p id="demo3"></p>

<script>
var person = {
    firstName  : "John",
    lastName   : "Smith",
    occupation : "Book editor",
    details : function(c) {
       return this.firstName + " " + this.lastName + " " + "-" +" " + this.occupation;
    }
};

document.getElementById("demo3").innerHTML = person.details();
</script>
