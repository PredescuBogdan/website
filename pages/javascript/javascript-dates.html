Definition

JavaScript dates can be written as numers or strings. Using the date object we can work with dates (years, months, days, hours, minutes, seconds, and milliseconds)

Syntax

new Date()

  Creates a Date object with current date and time:

  var now = new Date()

  alert(now)

Example

Displaying Dates

Code:

<p id="demo"></p>

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.toString();
</script>
