Definition

All HTML elements can be considered as boxes. When we talk about design and layout we use the term "box model". Using box model we can add a border to elements or add space between elemnts. A box model consists of: margins, borders, padding and the content. The padding clears an area around the content and the margin clears an area outside the border.

Syntax

div {

width: 320px;

padding: 10px;

border: 5px solid gray;

margin: 0;

}
