Definition

Using CSS we can make a HTML table better. Using CSS we can define the table color, width, borders, collapse borders, horizontal text alignment, vertical text alignment, table pading and so on. Width and height of a table is defined by the width and height properties. The border-collapse property sets whether the table borders are collapsed into a single border or separated. Table padding is used to control the space between the border and the content of a table.

Syntax

Table Borders

table, th, td {

border: 1px solid black;

}

Table Width and Height

table {

width: 100%;

}

th {

height: 50px;

}

Horizontal Text Alignment

th {

text-align: left;

}

Table Padding

td {

padding: 15px;

}
