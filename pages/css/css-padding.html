Definition

The space between the element border and the element content are called CSS padding properties. The padding clears an area around the content (inside the border) of an element. The background color of the element affects the padding. All padding like padding-top, padding-right, padding- bottom, padding-left can be changed separately with properties.

Syntax

p {

padding-top: 25px;

padding-right: 50px;

padding-bottom: 25px;

padding-left: 50px;

}

Or:

p {

padding: 25px 50px 25px 50px;

}

Example

Padding - Individual sides

Code:

<head>
<style>
.text {
    background-color: yellow;
}

.padding {
    padding-top: 25px;
    padding-right: 50px;
    padding-bottom: 25px;
    padding-left: 50px;
}
</style>
</head>
<body>

<p class="text">This is a simple paragraph with no specified padding.</p>

</body>
